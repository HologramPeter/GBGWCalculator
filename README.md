# GBGWCalculator – Game Simulator

A JavaScript-based simulator for **Gundam Breaker Mobile** that models game mechanics and keeps data updated automatically through web crawling.

## Overview

This project simulates gameplay systems from **Gundam Breaker Mobile**, allowing experimentation with builds, parts, and mechanics outside the game environment.  
Game data is automatically updated by crawling publicly available sources, ensuring the simulator remains synchronized with the latest content.

## Features

- Game mechanics simulation
- Automated data updates via web crawling
- Parts and build configuration testing
- Data-driven simulation architecture
- Modular JavaScript codebase

## Tech Stack

- JavaScript
- Web Crawling / Scraping
- JSON-based data models
- PHP
